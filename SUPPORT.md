# Support

## Bugs

Report reproducible bugs with the
[bug report form](https://github.com/echoVic/blade-deepseek/issues/new?template=bug_report.yml).
Include the Orca version, installation method, platform, operating mode, exact
reproduction steps, and redacted diagnostics.

## Feature Requests

Propose focused product changes with the
[feature request form](https://github.com/echoVic/blade-deepseek/issues/new?template=feature_request.yml).

## Documentation

Report missing, incorrect, or unclear documentation with the
[documentation form](https://github.com/echoVic/blade-deepseek/issues/new?template=documentation.yml).

## Usage Help

## Security

Report security vulnerabilities only through a
[private GitHub security advisory](https://github.com/echoVic/blade-deepseek/security/advisories/new).
Do not open a public Issue.

Before sharing any report or diagnostic output, remove API keys, tokens, private
source code, personal data, and sensitive logs.
