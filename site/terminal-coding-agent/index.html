Why it fits

Orca keeps the coding loop inside your terminal.

Local repository work

Search, read, edit, and verify files without moving code through a separate chat window.

Approval-aware tools

Reads can move quickly, while edits, shell commands, network calls, MCP tools, and subagents follow policy.

Long-running tasks

Persistent goals and resumable JSONL history help long refactors continue across turns and restarts.
