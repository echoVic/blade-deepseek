# Orca v0.1.8

TUI update preflight release.

## Changes

- Check for available updates before starting the interactive TUI.
- Stop before entering the TUI when a newer GitHub Release is available.
- Keep update-check failures non-blocking, so offline or transient GitHub API failures still allow the TUI to start.
- Remove the previous in-TUI background update notice path to avoid duplicate update behavior.

## Install

```bash
npm install -g @blade-ai/orca
orca --version
```

Direct binary install:

```bash
curl -fsSL https://raw.githubusercontent.com/echoVic/blade-deepseek/main/install.sh | sh
```
