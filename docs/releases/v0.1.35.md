# Orca v0.1.35

Patch release: bracketed paste support and textarea soft-wrap fix.

## Changes

- **Bracketed paste**: the TUI input now handles bracketed paste sequences
  (`\x1b[200~…\x1b[201~`), allowing multi-line content to be pasted without
  triggering premature submission or garbled rendering.
- **Soft-wrap rendering rewrite**: textarea line-wrap logic is rewritten to
  correctly account for wide characters and ANSI escape sequences, fixing
  inaccurate height calculation that could clip long input or leave blank gaps.

## Compatibility

No changes to CLI flags, JSONL event names, server-mode wire shapes, npm
package layout, or install commands.

## Install

```bash

orca --version
```

Direct binary install:

```bash
curl -fsSL https://orcaagent.dev/install.sh | sh
```

Pin a specific version:

```bash
curl -fsSL https://orcaagent.dev/install.sh | \
  INSTALL_DIR=/usr/local/bin ORCA_VERSION=0.1.35 sh
```
