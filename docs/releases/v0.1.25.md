# Orca v0.1.25

Bug fix release: inject current date into system prompt and fix mouse scroll behavior.

## Changes

- **Fix web search using wrong year**: Inject `Today's date` into the system prompt so the model knows the current date and uses the correct year in search queries, instead of guessing based on its training cutoff.
- **Fix mouse scroll in input area**: Remove the content-region-only restriction on mouse wheel events. Scrolling now works regardless of where the mouse cursor is positioned (content area, input box, or status bar), and scroll events never leak into the textarea.
