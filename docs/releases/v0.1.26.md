# Orca v0.1.26

Bug fix and improvement release: fix update check reliability and improve table rendering.

## Changes

- **Fix version update check**: The update check previously relied solely on the GitHub Releases API, which silently fails under rate limiting (403). Now uses npm registry (`registry.npmjs.org`) as the primary version source (no auth required, no rate limit), with GitHub API as fallback only when npm is unreachable.
- **Improve table rendering**: Completely rewrite table rendering with progressive degradation inspired by Codex CLI:
  - **Wide terminal**: Tables render at natural content width with no compression.
  - **Medium terminal**: Column widths are proportionally shrunk (largest columns shrink first), and cell content wraps at word boundaries across multiple visual lines.
  - **Narrow terminal**: When columns are compressed below 12 characters with 3+ columns, automatically degrades to key/value record mode for readability.
  - **Visual refresh**: Remove box-drawing borders, use bold `━` header separator, 2-char gap between columns, Cyan+Bold headers.
