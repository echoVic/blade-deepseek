# Orca v0.1.139

Patch release: isolate the child-agent provider-turn boundary so the
`agent_child` module keeps compatibility re-exports while model routing,
provider hook execution, provider call dispatch, child compaction, and
prompt-too-long retry/failure handling live in a focused module.

## Changes

- **Focused provider-turn module**: `child_agent_provider_turn.rs` now owns
  `ChildAgentProviderTurn`, `ChildAgentProviderErrorDecision`,
  `route_child_agent_model`, `run_child_agent_provider_turn`,
  `compact_child_agent_conversation_if_needed`, and
  `handle_child_agent_provider_error`.
- **Compatibility re-export**: existing `orca_runtime::agent_child` imports for
  child-agent provider-turn APIs remain available via re-export.
- **Agent-child shrink**: `agent_child.rs` no longer owns child model routing,
  pre/post model hook execution, direct provider calls, child compaction setup,
  or prompt-too-long retry/failure classification.
- **Ownership coverage**: a new architecture test locks the provider-turn
  boundary so the `agent_child` facade does not silently reacquire provider
  routing or error-handling details.

## Compatibility

No changes to CLI flags, JSONL event names, subagent tool schema, workflow child
agent calls, async subagent workers, TUI subagent execution, model override
behavior, provider response folding, tool-call extraction, tool-result folding,
cost tracking, approval policy semantics, MCP routing, external tool routing,
shell session protocol, command/exec wire shape, conversation storage, history
storage, npm package layout, or install commands.

This release is an internal ownership slice. Child-agent behavior for subagent
tool calls, workflow child agents, async subagents, TUI child-agent delegation,
provider bootstrap, model routing, pre/post model hooks, prompt-too-long
recovery, and turn-budget exhaustion is unchanged; the provider-turn boundary is
smaller and easier to keep aligned with the rest of the Codex/package-3-inspired
runtime split.

```bash

```

```bash

  INSTALL_DIR=/usr/local/bin ORCA_VERSION=0.1.139 sh
```
