# Orca v0.1.19

TUI user-input release.

## Changes

- Complete the interactive TUI answer loop for the model-visible `request_user_input` tool.
- Show structured questions and choices in the conversation, switch the status line to input-waiting mode, and send the next composer submission back as the tool answer.
- Keep headless and non-interactive behavior deterministic while allowing TUI turns to continue after the user answers.
- Add regression coverage proving a TUI turn waits for a user answer and then completes.

## Install

```bash
npm install -g @blade-ai/orca
orca --version
```

Direct binary install:

```bash
curl -fsSL https://orcaagent.dev/install.sh | sh
```
