# Orca v0.1.3

Input history and formatting release.

## Changes

- Use Up and Down in the idle input box to restore previous prompt entries.
- Preserve Up and Down cursor movement when editing multi-line prompts.
- Keep Ctrl+P and Ctrl+N as alternate prompt history shortcuts.
- Refresh shortcut help text for prompt history navigation.
- Apply repository-wide `cargo fmt`.

## Install

```bash
npm install -g @blade-ai/orca
orca --version
```

Direct binary install:

```bash
curl -fsSL https://raw.githubusercontent.com/echoVic/blade-deepseek/main/install.sh | sh
```
