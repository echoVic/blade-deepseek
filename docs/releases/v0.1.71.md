# Orca v0.1.71

Patch release: runtime compaction now lives in a dedicated module, further shrinking `lifecycle.rs`.

## Changes

- **Compaction boundary**: `RuntimeCompactionStep` and its hook/persistence logic now live in `crates/orca-runtime/src/compaction.rs`.
- **Thinner lifecycle module**: `lifecycle.rs` no longer owns the compaction step implementation.
- **Ownership guard**: runtime tests now keep compaction ownership pinned to the new module.

## Compatibility

No changes to CLI flags, config file shape, server-mode methods, npm package layout, install commands, JSONL event names, public realtime event names, or TUI display behavior.
