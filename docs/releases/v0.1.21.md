# Orca v0.1.21

TUI update-prompt bugfix release.

## Changes

- Treat Ctrl-C and Ctrl-D in the startup update prompt as an explicit quit with exit code 130 instead of continuing into the TUI.
- Keep the raw-mode update prompt left-aligned by writing CRLF line endings while the prompt is active.
- Keep release metadata aligned across Cargo, README, roadmap, site metadata, and release notes.

## Install

```bash
npm install -g @blade-ai/orca
orca --version
```

Direct binary install:

```bash
curl -fsSL https://orcaagent.dev/install.sh | sh
```
