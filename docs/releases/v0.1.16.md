# Orca v0.1.16

User-ergonomics release.

## Changes

- Add a model-visible `request_user_input` tool contract for structured clarification questions.
- Keep headless and non-interactive execution deterministic by returning a clear failure instead of blocking.
- Register the tool as read-only, state-rendered, and non-concurrent so future TUI answer handling can build on a stable tool surface.

## Install

```bash
npm install -g @blade-ai/orca
orca --version
```

Direct binary install:

```bash
curl -fsSL https://orcaagent.dev/install.sh | sh
```
