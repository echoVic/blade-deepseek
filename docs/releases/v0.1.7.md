# Orca v0.1.7

Update-check correctness release.

## Changes

- Fix update notifications to compare GitHub Releases against the installed Orca CLI version instead of the internal runtime crate version.
- Suppress false update notices when the installed CLI is equal to or newer than the latest GitHub Release.
- Show the same CLI version in the TUI welcome screen that `orca --version` reports.
- Keep DeepSeek V4 context budgeting aligned to the 1M-token model window.

## Install

```bash
npm install -g @blade-ai/orca
orca --version
```

Direct binary install:

```bash
curl -fsSL https://raw.githubusercontent.com/echoVic/blade-deepseek/main/install.sh | sh
```
