# Orca v0.1.18

Skills-injection release.

## Changes

- Inject explicitly mentioned Markdown skills into the model context when a prompt references a skill id such as `$debugging`.
- Keep skill injection scoped to user and project skills discovered from `$ORCA_HOME/skills`, `~/.orca/skills`, and project `.orca/skills`.
- Add history coverage proving headless `orca exec` records the injected skill instructions in the system prompt.
- Document skill tools and update the roadmap baseline to reflect the v0.1.18 skills state.

## Install

```bash
npm install -g @blade-ai/orca
orca --version
```

Direct binary install:

```bash
curl -fsSL https://orcaagent.dev/install.sh | sh
```
