# Orca v0.1.175

Patch release: add read-time output caps to server-mode `command/exec/read`.

## Changes

- **Command exec read caps**: server-mode clients can now send
  `outputBytesCap` on `command/exec/read` to tighten the active streaming
  process output cap before polling output.
- **Pre-drain safety**: the cap is applied before the server's normal active
  process pre-dispatch drain, so output that becomes available right before the
  read request is still bounded by the read budget.
- **UTF-8-safe deltas**: capped read-triggered `command_exec_output_delta`
  events reuse the existing UTF-8 boundary handling and set `capReached: true`
  when the cap is reached.
- **Non-widening behavior**: read caps can narrow an active process output cap,
  but they do not widen a stricter cap that was already set on `command/exec`.

## Compatibility

No changes to CLI flags, slash commands, model/provider configuration, MCP
configuration, hook schema, workflow scripts, npm package layout, install
commands, or existing JSONL event names.

Existing `command/exec/read` requests without `outputBytesCap` keep the same
behavior. Existing `command/exec` start-time `outputBytesCap`,
`command/exec/write`, `command/exec/resize`, `command/exec/terminate`,
`shell/read`, `shell/capabilities`, and PTY behavior is unchanged.

Example request:

```json
{
  "id": "cmd-read",
  "method": "command/exec/read",
  "params": {
    "processId": "process-1",
    "timeoutMs": 5000,
    "outputBytesCap": 4096
  }
}
```

Read-triggered output continues to arrive as `command_exec_output_delta` events
and the Codex-style `command/exec/outputDelta` notification shape:

```json
{
  "event": "command_exec_output_delta",
  "method": "command/exec/outputDelta",
  "processId": "process-1",
  "stream": "stdout",
  "delta": "partial output",
  "capReached": true
}
```

## Install

```bash
npm install -g @blade-ai/orca
orca --version
```

Direct binary install:

```bash
curl -fsSL https://orcaagent.dev/install.sh | sh
```

Pin a specific version:

```bash
curl -fsSL https://orcaagent.dev/install.sh | \
  INSTALL_DIR=/usr/local/bin ORCA_VERSION=0.1.175 sh
```
