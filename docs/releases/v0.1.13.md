# Orca v0.1.13

Context-reliability release.

## Changes

- Add `[model_runtime]` config for model context-window and auto-compact token-limit overrides.
- Route model runtime overrides through headless, TUI, child-agent, workflow, and manual compaction paths.
- Make context budget hints use the effective runtime limit for the routed model.
- Show model runtime limits in `/config show`.

## Config

```toml
[model_runtime]
context_window = 128000
auto_compact_token_limit = 96000
```

## Install

```bash
npm install -g @blade-ai/orca
orca --version
```

Direct binary install:

```bash
curl -fsSL https://orcaagent.dev/install.sh | sh
```
