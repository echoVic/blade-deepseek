# Orca v0.1.2

Workflow namespace correction release.

## Changes

- Resolve named project workflows from `.orca/workflows/`.
- Resolve named user workflows from `~/.orca/workflows/`.
- Remove `.claude/workflows/` compatibility from Orca workflow lookup.
- Update website and README workflow copy to describe Orca-native workflow directories.

## Install

```bash
npm install -g @blade-ai/orca
orca --version
```

Direct binary install:

```bash
curl -fsSL https://raw.githubusercontent.com/echoVic/blade-deepseek/main/install.sh | sh
```
