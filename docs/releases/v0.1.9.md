# Orca v0.1.9

Codex-style interactive update prompt release.

## Changes

- Replace the v0.1.8 hard update gate with a startup update prompt modeled after Codex CLI.
- Let users choose `Update now`, `Skip`, or `Skip until next version` before entering the TUI.
- Run `npm install -g @blade-ai/orca@latest --registry https://registry.npmjs.org` only when the user selects `Update now`.
- Persist `Skip until next version` in the Orca home update cache so the same release is not prompted again.
- Keep update-check failures non-blocking.

## Install

```bash
npm install -g @blade-ai/orca
orca --version
```

Direct binary install:

```bash
curl -fsSL https://raw.githubusercontent.com/echoVic/blade-deepseek/main/install.sh | sh
```
