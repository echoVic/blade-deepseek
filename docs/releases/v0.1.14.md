# Orca v0.1.14

User-ergonomics release.

## Changes

- Add a `/config show` runtime posture summary.
- Show approval mode, filesystem posture, network posture, history mode, context limits, tool truncation, and workflow agent limits in one scan-friendly section.
- Keep existing detailed config values below the summary.

## Install

```bash
npm install -g @blade-ai/orca
orca --version
```

Direct binary install:

```bash
curl -fsSL https://orcaagent.dev/install.sh | sh
```
