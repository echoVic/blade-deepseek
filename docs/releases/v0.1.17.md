# Orca v0.1.17

Skills-system release.

## Changes

- Add `list_skills` and `read_skill` tools for Markdown skill discovery.
- Discover user skills from `$ORCA_HOME/skills/*/SKILL.md` or `~/.orca/skills/*/SKILL.md`.
- Discover project skills from `.orca/skills/*/SKILL.md` while keeping project skill paths inside the workspace.
- Parse simple frontmatter `name` and `description` fields and return the Markdown body as instructions.

## Install

```bash
npm install -g @blade-ai/orca
orca --version
```

Direct binary install:

```bash
curl -fsSL https://orcaagent.dev/install.sh | sh
```
