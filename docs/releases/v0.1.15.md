# Orca v0.1.15

User-ergonomics release.

## Changes

- Add fuzzy fallback matching for TUI `@mention` file candidates.
- Preserve existing prefix completion behavior when direct matches exist.
- Cap fuzzy traversal and displayed candidates so large workspaces stay responsive.

## Install

```bash
npm install -g @blade-ai/orca
orca --version
```

Direct binary install:

```bash
curl -fsSL https://orcaagent.dev/install.sh | sh
```
