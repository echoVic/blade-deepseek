# Orca v0.1.4

Persistent goal mode release.

## Changes

- Add Codex-style `/goal` mode for long-running objectives in the TUI.
- Persist goals by saved session id in `$ORCA_HOME/goals_1.json` or `~/.orca/goals_1.json`.
- Add `/goal` commands to show, create, edit, pause, resume, and clear a session goal.
- Expose the `update_goal` tool during goal turns so the agent can mark goals complete or blocked.
- Continue active goals automatically until completion, interruption, budget stop, or the continuation cap.
- Document goal mode behavior, persistence, harness contract details, and production roadmap notes.

## Install

```bash
npm install -g @blade-ai/orca
orca --version
```

Direct binary install:

```bash
curl -fsSL https://raw.githubusercontent.com/echoVic/blade-deepseek/main/install.sh | sh
```
