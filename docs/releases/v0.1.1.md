# Orca v0.1.1

Packaging correction release.

## Changes

- Switch npm distribution to the Codex-style alias model.
- Keep `@blade-ai/orca` as the only package name published for new Orca tarballs.
- Publish platform binaries as prerelease versions such as `0.1.1-darwin-arm64`.
- Keep install usage unchanged:

```bash
npm install -g @blade-ai/orca
orca --version
```
