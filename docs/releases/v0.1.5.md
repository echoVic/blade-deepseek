# Orca v0.1.5

DeepSeek V4 context window correction.

## Changes

- Update model context budgeting for DeepSeek V4 Flash and Pro to use the 1M-token context window.
- Align automatic compaction defaults with the V4 context window instead of the old 128K fallback.
- Update context-window documentation and install examples for the new patch release.

## Install

```bash
npm install -g @blade-ai/orca
orca --version
```

Direct binary install:

```bash
curl -fsSL https://raw.githubusercontent.com/echoVic/blade-deepseek/main/install.sh | sh
```
