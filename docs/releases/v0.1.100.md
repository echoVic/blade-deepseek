# Orca v0.1.100

Patch release: TUI experience polish — hardened inline scrolling, background
memory extraction, and a live activity indicator.

## Changes

- **Scroll hardening**: inline scrolling now uses ratatui's
  `unstable-rendered-line-info` to detect real overflow, so wheel/PageUp on a
  transcript that still fits keeps auto-follow armed, while an overflowing
  transcript lets the user break away and read back.
- **Auto-scroll locking**: auto-follow only disarms when there is genuinely
  scrollable content, fixing the "state changed but the view stayed pinned"
  behavior.
- **CJK-aware wrap height**: mixed-width text now computes wrapped-line height
  correctly, removing the scroll offset drift that CJK characters caused.
- **Background memory extraction**: the memory-extraction LLM round-trip moved
  off the render thread so the UI no longer blocks while it runs.
- **Activity indicator**: a new activity bar plus a refreshed status-bar layout
  surface live progress.
- **Mouse-scroll debounce**: inertial trackpad scrolling is ignored for a short
  grace period right after a turn completes, so momentum from the last gesture
  no longer yanks the freshly settled transcript.
- **Docs/site copy**: README and site marketing copy refreshed.

## Compatibility

No changes to CLI flags, JSON protocol method names, server event names,
permission profile TOML shape, install commands, or npm package layout. The
`unstable-rendered-line-info` ratatui feature is an internal rendering
dependency only.

## Install

```bash
npm install -g @blade-ai/orca
orca --version
```

Direct binary install:

```bash
curl -fsSL https://orcaagent.dev/install.sh | sh
```

Pin a specific version:

```bash
curl -fsSL https://orcaagent.dev/install.sh | \
  INSTALL_DIR=/usr/local/bin ORCA_VERSION=0.1.100 sh
```
