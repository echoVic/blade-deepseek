# Orca v0.1.12

Context-reliability release.

## Changes

- Add a configurable tool output truncation policy under `[tools]`.
- Preserve the historical byte truncation default while adding token-budget mode for large tool outputs.
- Token truncation adds an explicit warning with original token and line counts before the compacted output.
- Route configured truncation through runtime and TUI tool execution paths, including shell and external tools.
- Show the effective output truncation policy in `/config show`.

## Config

```toml
[tools]
output_truncation = { mode = "tokens", limit = 2000 }
```

## Install

```bash
npm install -g @blade-ai/orca
orca --version
```

Direct binary install:

```bash
curl -fsSL https://orcaagent.dev/install.sh | sh
```
