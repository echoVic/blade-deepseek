## 二、任务清单

### 2.1 专用代理类型（优先级：P0）

**目标**: 实现针对特定任务优化的子代理类型

**代理类型**:
1. **CodeReviewer** - 代码审查专家
2. **TestWriter** - 测试编写专家
3. **Debugger** - 调试专家
4. **Documenter** - 文档编写专家
5. **General** - 通用代理（默认）

**实现要点**:
- 每种类型有专门的系统提示
- 不同的工具访问权限
- 针对性的审批策略

### 2.2 Worktree 隔离（优先级：P1）

**目标**: 支持在独立的 git worktree 中运行子代理

**功能**:
- 自动创建临时 worktree
- 并行子代理零冲突
- 无变更自动清理
- 有变更保留供审查

### 2.3 模型选择（优先级：P1）

**目标**: 支持为不同子代理选择不同模型

**功能**:
- 模型参数传递
- 成本优化（简单任务用 flash，复杂任务用 reasoner）
- 配置继承

### 2.4 实际 Agent Loop 集成（优先级：P0）

**目标**: 将后台执行与真实的 agent loop 连接

**功能**:
- 调用 run_agent_loop
- 进度实时更新
- 输出文件持续写入
- 完整的错误处理
